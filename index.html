Inazuma Football Game

An exciting command-line interface (CLI) game featuring iconic moves from the Inazuma Eleven series. Challenge the computer in a single-player showdown!

About the Game

This game is a CLI-based football game inspired by the popular Inazuma Eleven series. It includes many of the series' special moves and allows players to compete against a computer opponent. Whether you're a fan of the anime or just love football games, this project is perfect for you!

Key Features

  • Single-player mode against a computer opponent.
  • Incorporates popular moves from the Inazuma Eleven series.
  • Simple and intuitive command-line interface.
